Unsupervised Deep Learning: Self Organising Maps

Self Organising Map can help with reducing multidimensionality of input data f.e. to visualise them easier

K-Means clustering - grupowanie danych metoda k-srednich
	https://www.statystyka.az.pl/analiza-skupien/metoda-k-srednich.php

	step 1: choose the number of clusters
	step 2: select random points, initial centroids
	step 3: assign each data point to the closest centroid
	step 4: compute and place the new centroid of each cluster
	step 5: reassign each data point to the new closest centroid
		if any reassignment took place, go to STEP 4
